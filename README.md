# Twitter Sentiment Analysis App

This project predicts whether a tweet has positive or negative sentiment using Machine Learning.

## Tech Stack
- Python
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression
- Streamlit
- GitHub + Streamlit Cloud

## Features
- Real-time sentiment prediction
- Clean web interface
- Deployed online

## Run Locally
pip install -r requirements.txt
streamlit run app.py
